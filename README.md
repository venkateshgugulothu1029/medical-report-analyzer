# 🩺 Medical Report Analyzer (AI for Healthcare)

## 🚀 Overview
AI-powered application that analyzes medical reports and provides insights using LLMs.

## 🧠 Features
- Upload PDF medical reports
- Extract and analyze text
- Generate:
  - Summary
  - Key observations
  - Health risks

## 🛠️ Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI
- PyPDF
